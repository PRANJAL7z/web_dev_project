NEW COLOR SCHEME

🟢 GREEN DOTS = Low Danger (Low crime scores)
   - Cities with lower crime severity
   - Safer regions

🟡 YELLOW/ORANGE DOTS = Medium Danger (Medium crime scores)
   - Cities with moderate crime severity
   - Average risk areas

🔴 RED DOTS = High Danger (High crime scores)
   - Cities with highest crime severity
   - High-risk areas requiring attention

⬛ BLACK LINE = Regression Line
   - Shows the prediction trend
   - Easy to see against colored dots

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

HOW IT WORKS

The color is automatically calculated based on the crime score:

• Score 0.0 - 0.33 → 🟢 GREEN (Low danger)
• Score 0.34 - 0.66 → 🟡 YELLOW/ORANGE (Medium danger)
• Score 0.67 - 1.0 → 🔴 RED (High danger)

Each dot shows:
- City name (above dot)
- Crime score value (below city name)
- Color indicating danger level
